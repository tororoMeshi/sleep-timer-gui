# 🎵 Sleep Timer with Pause

![icon](../icon.png)

**指定時間に音楽を停止し、PCをスリープするクロスプラットフォームタイマー**

---

## 🖥 対応OS

- 🪟 Windows 10 / 11
- 🐧 Linux（Ubuntu, Debian, Fedora, Arch など AppImage 対応）

---

---

## 🔧 主な機能

✅ タイマー（○分後 / 指定時刻）  
✅ 音楽停止（メディアキー）  
✅ スリープ  
✅ エラー時ポップアップ＆ログ  
✅ 前回設定の自動復元  
✅ 自動起動（スタートアップ）設定  
✅ タスクトレイ常駐
